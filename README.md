# EchoVanta Core

This repository contains utilities for working with EchoVanta. A simple terminal user interface (TUI) is provided to browse files.

## Using the TUI

Run the following command from the repository root to launch the interface:

```bash
python terminal_ui.py
```

Use the **up** and **down** arrow keys to choose a file and press **Enter** to open it. Press **q** to exit the interface.
